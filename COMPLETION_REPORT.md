### ⚠️ Что нужно сделать перед запуском:

1. **Настроить переменные окружения:**
   ```bash
   ./setup_local.sh
   # или
   cp .env.example .env
   # Отредактировать .env
   ```

2. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настроить базу данных:**
   ```bash
   createdb traffk_db
   alembic upgrade head
   ```

4. **Получить BOT_TOKEN:**
   - Откройте @BotFather в Telegram
   - Создайте бота командой /newbot
   - Скопируйте токен в .env

5. **Получить ADMIN_IDS:**
   - Откройте @userinfobot в Telegram
   - Скопируйте ваш ID в .env

## 📝 Следующие шаги

### Для локального тестирования:

```bash
# 1. Настройте окружение
./setup_local.sh

# 2. Проверьте конфигурацию
python3 scripts/check_config.py

# 3. Установите зависимости
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 4. Настройте БД
createdb traffk_db
alembic upgrade head

# 5. Запустите бота
python bot.py
```

### Для развертывания на VPS:

```bash
# 1. Создайте архив
tar --exclude='.git' --exclude='venv' --exclude='.env' \
    -czf traffk-deploy.tar.gz .

# 2. Загрузите на VPS
scp traffk-deploy.tar.gz user@vps-ip:/tmp/

# 3. На VPS запустите
ssh user@vps-ip
cd /tmp && tar -xzf traffk-deploy.tar.gz
cd Traffk_mvp_beta_ai_cursor
sudo ./scripts/deploy.sh

# 4. Настройте .env
sudo nano /opt/traffk/.env

# 5. Запустите
sudo systemctl start traffk-bot
```
