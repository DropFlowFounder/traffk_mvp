## 🎯 Ключевые особенности

### Безопасность
- Ручное управление эскроу администратором
- Whitelist для админ-команд
- Telegram ID как идентификатор
- Защита чувствительных данных

### Функциональность
- Роли: Рекламодатель, Исполнитель, Оба
- Статусы заданий: Черновик → Опубликовано → В работе → На проверке → Завершено
- Система proof с загрузкой фото
- Споры и арбитраж
- Комиссия 5% с каждой сделки

### Технические детали
- Асинхронный код (aiogram 3.x)
- PostgreSQL с asyncpg
- Миграции через Alembic
- Логирование в файл и консоль
- Docker для контейнеризации
