# 📜 Скрипты для развертывания

## check_config.py

Проверяет конфигурацию перед запуском бота.

**Использование:**
```bash
python scripts/check_config.py
```

Проверяет:
- Наличие файла .env
- Наличие обязательных переменных окружения
- Формат ADMIN_IDS
- Подключение к базе данных

## setup_env.sh

Интерактивный скрипт для настройки переменных окружения.

**Использование:**
```bash
chmod +x scripts/setup_env.sh
./scripts/setup_env.sh
```

Запрашивает значения для:
- BOT_TOKEN
- DATABASE_URL
- ADMIN_IDS
- SENTRY_DSN (опционально)
- ENVIRONMENT

## deploy.sh

Автоматический скрипт для развертывания на VPS.

**Использование:**
```bash
chmod +x scripts/deploy.sh
sudo ./scripts/deploy.sh
```

**Требования:**
- Запуск от root или с sudo
- Ubuntu/Debian система

**Что делает:**
1. Устанавливает системные зависимости
2. Создает пользователя приложения
3. Настраивает PostgreSQL
4. Устанавливает Python зависимости
5. Создает systemd сервис

**Внимание:** После запуска скрипта необходимо отредактировать `/opt/traffk/.env` и указать реальные значения BOT_TOKEN и ADMIN_IDS.
